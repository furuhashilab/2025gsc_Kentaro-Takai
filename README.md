## Abstract
本研究は、Minecraft を用いて青山学院大学相模原キャンパスの建物を再現し、教育的活用につながる可能性を検討するものである。近年、ゲームを用いた学習支援が広がりつつあり、とくに Minecraft は空間を歩き回りながら理解できる学習環境として注目されている。しかし、子どもや学生が実際に訪れる可能性のある「大学キャンパス」をデジタルツインとして再構築した例は少なく、外観だけでなく内部空間まで含めて再現した研究はほとんど見られない。そこで本研究では、相模原キャンパス B棟を対象に、現地で記録可能な範囲の情報を用いて Minecraft 上に再現し、その過程で生じた課題や判断プロセスを整理することを目的とする。

先行研究では、教育分野における Minecraft 活用の有効性、PLATEAU を用いたキャンパス外観モデルの公開、Minecraft を用いたまちづくりワークショップの実践などが報告されているが、内部空間の精密な再現プロセスを扱った研究はない。本研究では、外観の再現手法、内部空間の取捨選択、写真や歩測による寸法推定など、再現のための具体的な手法を明確化した。また、PLATEAU データの利用可能性を検討しつつ、教育的用途に耐える精度を確保するためには手作業が有効であることを確認した。完成後は、再現した空間を活用した体験型イベント（例：防災学習など）を実施し、学習効果を検証する予定である。本研究は、大学キャンパスのデジタルツインを教育用途に応用するための基礎的知見を提供する。


---

## Introduction
大学キャンパスは複雑な空間構造を持ち、新入生や来訪者が建物内の位置関係や動線を直感的に把握することは難しい。こうした背景から、空間を体験的に理解できる教材の必要性が高まっている。近年では教育分野にゲームを取り入れる事例が増えており、Minecraft は学習者が能動的に空間を探索できる点から注目を集めている。しかし、学校や自治体での導入例が増える一方で、「実際に子どもや学生が訪れる可能性のある大学キャンパス」を対象に、外観と内観の双方をデジタルツインとして再現した研究はほとんど存在しない。

また、先行研究としては、Minecraft を空間学習に用いた事例、PLATEAU による相模原キャンパス外観モデルの公開、Minecraft によるまちづくりワークショップの実践が報告されているが、建物内部の再現プロセスや、教育的活用に適した空間構成の検討は十分に行われていない。そこで本研究では、青山学院大学相模原キャンパス B棟を対象に、現地で取得可能な情報をもとに Minecraft 上で内部空間を再現し、その過程での課題や判断基準を整理することを目的とする。最終的には、この再現空間を活用した体験型イベント（例：防災学習など）につなげる基礎を構築する。

---

## Methods
本研究では、青山学院大学相模原キャンパス B棟の再現を目的として、①データ検討、②手作業による再現、③イベント計画の三段階で進めた。まずデータ検討では、国土交通省 PLATEAU 3D 都市モデル（相模原市データ）の利用可能性を検証した。外観モデルを Minecraft へインポートする方法を試したが、変換過程でジャギーが生じ、建物形状が歪む問題が確認された。そのため、教育教材として利用するには十分な精度を確保できないと判断し、PLATEAU データの利用は断念した。

続いて、手作業による再現では、Minecraft Java Edition を用い、現地で取得可能な範囲の情報（写真、動画、歩測、構造の観察）をもとに B棟の外観および内部空間を構築した。廊下幅、階段位置、柱の本数、部屋の配置などは現地写真を参照しながらブロック単位で再現し、視界に依存する部分は複数回の現地観測で補った。また、上階の構造など観察が困難な部分については、先行研究である柴山伊吹氏（2022年度卒論）の成果を参考にして補完した。

最後に、イベント計画として、B棟完成後に研究室内で実施予定の「体験型イベント」の設計を行った。参加者には、避難経路探索やシミュレーション訓練などを体験してもらい、Minecraft 上で再現した空間が教育・防災学習において有効かを検証する予定である。

## Discussion（予定）
本研究で構築した Minecraft 上の相模原キャンパス B棟モデルは、教育や防災学習における体験型教材として活用できる可能性を示唆している。実際に歩き回りながら空間を把握できるため、図面や文章では理解しづらい動線や構造を直感的に理解できる点は、既存教材にはない特徴である。また、PLATEAU データと比較すると、手作業による再現は大幅な作業量を必要とする一方で、教育利用に求められる視覚的精度や空間把握のしやすさを確保しやすいことが確認された。

一方で、内部空間の再現には、現地観察で得られる情報量の限界や、寸法の推定に伴う不確実性が残る。また、Minecraft の特性上、1m ブロック単位で構築されるため、細かな造作の表現には限界がある。今後の課題として、VR/AR との連携による没入感の向上や、多人数が同時参加できるイベント設計などが挙げられる。さらに、A棟・C棟を含めたキャンパス全体のモデル化を進めることで、防災教育や新入生オリエンテーションなど、多様な用途への発展が期待される。

---

## Conclusion（予定）
本研究は、Minecraft を用いて青山学院大学相模原キャンパス B棟を再現し、教育・防災学習に活用するための基盤を構築した。外観・内観の双方を手作業で再現する過程を整理し、PLATEAU データでは補えない精度や柔軟性を確保できることを明らかにした点は、本研究の重要な成果である。再現後は、体験型イベントを通じて学習効果を検証する予定であり、本研究はキャンパスのデジタルツインを教育用途に応用する第一段階として位置づけられる。

今後は、他棟の再現や VR/AR の導入、多人数参加型イベントの設計などを通じ、より実践的な教育コンテンツへの発展を目指す。本研究が、ゲーム空間を活用した学習支援の新たな可能性を示す一例となることを期待する。
