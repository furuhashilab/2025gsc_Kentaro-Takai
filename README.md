## Abstract
本研究は、防災教育への活用を前提とした建物再現の作成プロセスを明らかにすることを目的とし、青山学院大学相模原キャンパスB棟を対象に、Minecraft 上での建物空間の再現を行った。大学キャンパスのような大規模建築物においては、非常階段や避難経路、消火器などの防災設備の位置関係を平面図や文章のみで理解することは難しく、体験的に空間を把握できる教材が求められている。

近年、Minecraft をはじめとするゲーム型学習環境が教育分野で注目されているが、防災教育を前提として実在する建物をどのように再現すべきか、その設計上の判断や作成プロセスを整理した研究は十分に行われていない。そこで本研究では、防災教育を目的とした共同研究の一部として、相模原キャンパスB棟の建物空間を Minecraft 上に再現し、その過程で生じた課題や工夫点を整理した。

再現にあたっては、当初 PLATEAU データの活用を試みたが、データ変換時の不具合により手作業での再現へと方針を転換し、現地調査、写真・動画資料、既存研究をもとに建物を構築した。防災教育に必要な空間要素を優先的に再現し、教室内装などの細部は抽象化する設計を採用した。また、再現した建物を用いて体験型ゲームを実施し、ゲーム前後で防災設備に関する理解度テストを行った。

その結果、ゲーム実施後には参加者の理解度が向上し、防災設備の位置関係に対する認識が高まったことが確認された。これらの結果から、防災教育を目的とした場合、建物内部の完全再現は必ずしも必要ではなく、目的に応じた再現範囲の選択と視覚的表現の工夫が重要であることが示唆された。

## Introduction
大学キャンパスは複雑な内部構造を持ち、新入生や来訪者にとって建物内の位置関係や動線を直感的に把握することは容易ではない。特に防災の観点からは、非常階段や避難経路、消火器などの安全設備の位置を正しく理解しておくことが重要であるが、平面図や文章のみを用いた学習では、実際の空間構造を十分にイメージできないという課題がある。

近年、Minecraft をはじめとするゲーム型学習環境は、仮想空間を自由に移動しながら対象を体験的に理解できる点から、教育分野や防災分野において注目を集めている。特に防災教育においては、避難経路や非常階段、消火器などの安全設備の位置を「空間として理解すること」が重要であり、ゲーム空間はその理解を支援する教材としての可能性を有している。

一方で、防災教育への活用を前提として実在する建物をゲーム空間上に再現する場合、建物をどのように工夫して再現すべきか、また非常階段や消火器といった防災設備の位置を学習者に把握してもらうために、どのような視覚的表現が適切であるかといった設計上の判断が求められる。しかし、既存の取り組みの多くは建物の外観や簡略化された構造の再現にとどまっており、大学キャンパスのような大規模建築物について、内部空間を含めた再現プロセスや、その際に生じる課題や判断基準を整理した研究は多くない。

本研究では、青山学院大学相模原キャンパスB棟を対象とし、防災教育に用いることを前提として、同建物を Minecraft 上に再現する過程に着目する。再現にあたっては、当初 PLATEAU データの活用を試みたものの、データ変換時の不具合などの問題が生じたため、最終的には現地調査や動画記録、手作業による再現へと方針を転換した。本研究の目的は、このような試行錯誤の過程を通じて、実空間をゲーム空間へ再構成する際に生じた課題や、それに対する取り組み、再現範囲および表現方法に関する判断プロセスを整理することである。

なお、本研究は共同研究の一部として実施されており、共同研究全体では、再現した相模原キャンパスB棟を用いて、防災要素を取り入れた体験型ゲームを実施している。本論文では、そのような防災利用を前提とした文脈のもとで行われた建物再現に焦点を当て、筆者が担当した空間設計および再現手法について報告する。

## Methods
**1. 研究対象**

本研究では、青山学院大学相模原キャンパスB棟を研究対象とし、同建物を Minecraft（Java Edition）上に再現した。
B棟は複数階から構成され、教室、研究室、廊下、階段など多様な内部空間を有する大学施設である。また、防災教育への活用を前提とした共同研究の対象建物であることから、本研究における建物再現の対象として適切であると判断した。

当初は相模原キャンパス全体の再現を計画していたが、建物数および作業量が膨大であること、さらに内部空間まで含めた再現には多くの時間と労力を要することから、最終的にB棟のみに対象を限定した。

**2. 再現に用いた資料およびデータ**
	
建物再現にあたっては、以下の資料およびデータを用いて空間情報を収集した。

* 教授より提供されたキャンパス公式配置図

* 現地での歩測調査

* 現地における写真および動画撮影

これらを組み合わせることで、建物の平面構成、移動経路、階段位置などを把握した。

また、現地での視認が困難であった上層部や一部外観形状については、先行する卒業研究（柴山伊吹，2022）で作成された建物データを補完資料として参照した。

**3. 再現スケールと基本方針**

Minecraft 上での再構成にあたっては、
1ブロック＝1m
を基本スケールとして採用した。

再現方針としては、防災教育への活用を前提とし、建物内のすべての要素を忠実に再現するのではなく、防災行動に直接関係する空間情報を優先的に再現する方針とした。

**4. 外観の再現方法**

外観の再現は、現地で撮影した写真および動画、先行する卒業研究（柴山伊吹，2022）をもとに、Minecraft 上で手作業によって行った。

当初は、Project PLATEAU による3D都市モデルを Minecraft ワールドへ変換する手法を検討したが、変換後のモデルにおいて形状の乱れや段差（ジャギー）が発生し、建物形状の把握や内部空間との整合性を確保することが困難であると判断した。

そのため、PLATEAU データの直接利用は断念し、手作業による再現へと方針を転換した。

**5. 内部空間の再現方法**

内部空間の再現にあたっては、防災行動との関係性を基準として、再現の精度を調整した。


A. 忠実に再現した要素

以下の要素については、防災時の行動や移動に直接関わることから、可能な限り実空間に忠実に再現した。

* 主要廊下の幅
* 非常階段の位置
* 避難動線（廊下・階段の接続関係）

これらの要素は、現地での歩測結果および写真・動画をもとに配置した。


B. 抽象化した要素

一方で、教室内部の机や椅子の配置、細部の備品など、防災行動との関連性が低い要素については、再現の負荷を考慮し、抽象化または簡略化を行った。

これにより、防災教育に必要な空間理解を損なうことなく、効率的な建物再現を行うことを目指した。



**6. 防災利用を前提とした表現上の工夫**

本研究は、防災教育を目的とした共同研究の一部として実施されている。そのため、ゲーム空間内で防災設備の位置を把握しやすくするための表現上の工夫を行った。

具体的には、消火器の設置位置には、ChatGPT を用いて生成した消火器の画像をテクスチャとしてブロックに適用し、現実空間における消火器の存在を視覚的に強調した。これにより、単なるオブジェクト配置ではなく、「ここに消火器がある」という情報を即座に認識できる表現とした。

また、非常階段や避難経路の入口についても、ChatGPT により生成した非常用出口を示すテクスチャを作成し、該当箇所に貼り付けることで、建物内における避難方向や安全な移動経路が分かりやすくなるよう工夫した。

これらの表現は、建物の形状を忠実に再現することよりも、防災行動に必要な空間情報を優先的に伝えることを目的として設計されており、防災教育向けのゲーム空間としての可読性および理解しやすさを高めることを意図している。

**7. 評価方法**

作成した Minecraft ワールドは、青山学院大学の学生を対象とした防災要素を取り入れた体験型ゲームにおいて実際に使用された。

評価方法として、イベント参加者に対し、ゲーム実施前（Before）およびゲーム実施後（After）に同一内容の防災理解度テストを実施した。
本テストは、青山学院大学相模原キャンパスB棟における防災設備の位置理解および空間的関係性の把握を測定することを目的とした。

テストは全5問で構成し、各設問を20点とする100点満点で評価した。設問は以下の観点に基づいて作成した。

* 非常階段が1階のどの施設につながっているかの理解

* 実在する非常口の位置の把握

* 消火器の典型的な設置位置に関する理解

* 避難器具（救助袋・避難はしご）の設置傾向の理解

* 複数の防災設備間における位置関係の総合理解

問題形式は単一選択および複数選択式とし、暗記的な知識ではなく、建物内を実際に歩いた経験によって得られる空間的理解が反映されるよう設計した。

テストは Google フォームを用いて実施し、ゲーム開始前およびゲーム終了後に同一の問題を提示した。

**8. ワールド実装環境と Education Edition への移行**

本研究における建物内部空間の再現は、Minecraft JAVA Edition 上で行った。
JAVA Edition はブロック操作や建築表現の自由度が高く、
実際の建物構造を詳細に再現するうえで適しているため、
初期段階のワールド制作環境として採用した。

一方で、本研究の成果を教育現場で活用することを想定した場合、
有料ソフトである JAVA Edition のみを前提とすることは、
利用者のアクセス性という観点で課題がある。
そこで、学校アカウントがあれば回数制限付きで無償利用が可能な
Minecraft Education Edition 上でも
本研究で作成した建物内部空間を利用可能な状態にすることを目標とした。

しかし、JAVA Edition で作成したワールドデータを
Education Edition に直接移行することは困難であったため、
Bedrock Edition を介した移行手法を検討した。
その結果、ワールド全体を移行するのではなく、
建築物単位でデータを切り出し、
Education Edition 上の新規ワールドに配置する方法を採用した。

最終的に、Education Edition 上で利用可能な
ワールドデータ（.mcworld 形式）を作成し、
当該ファイルをインポートすることで、
利用者が建物内部を閲覧・移動できる状態を構築した。
この方法により、JAVA Edition を利用しない場合でも、
Minecraft Education Edition を用いて
本研究で再現した建物内部空間を体験可能とした。

**1. B棟外観の再現結果**

Minecraft 上において、青山学院大学相模原キャンパスB棟の外観を再現した。
建物全体のボリューム、正面中央部の吹き抜け構造、左右のウイング部の配置関係を
Minecraft のブロック構造により表現した。

外観再現にあたっては、建物の寸法比および階層構成を優先し、
細部の意匠については簡略化を行った。

B棟の内部空間については、吹き抜け空間、主要な廊下、非常階段など、防災行動に関係する空間要素を中心に再現した。

一方で、教室内の机配置や内装の細部については、全体の防災理解に直接関与しないと判断し、抽象化を行った。これにより、内部空間全体の把握を妨げず、移動や避難行動に必要な構造が認識できる形で表現された。

**3. 防災設備の視覚的表現**

作成した Minecraft ワールド内では、消火器および非常階段といった防災設備が、他の構造物と視覚的に区別可能な形で配置された。

また、非常階段および避難経路の入口についても、周囲の建築要素とは異なる視覚表現が付与され、避難方向や安全な移動経路が直感的に判別できる状態となった。

これらの表現により、ゲーム空間内において防災設備が単なる装飾要素ではなく、行動判断に寄与する空間情報として機能していることが確認された。

**4. 防災理解度テストの結果**

作成した Minecraft ワールドを用いた防災イベントにおいて、参加者12名を対象に防災理解度テストを実施した。

ゲーム開始前（Before）のテストでは、平均得点は23.3点、中央値は20点であり、得点は0点から80点の範囲に分布していた。この結果から、多くの参加者が防災設備の位置関係や空間構造を十分に把握できていない状態であったことが確認された。

一方、ゲーム終了後（After）のテストでは、平均得点は75点、中央値は80点となり、得点範囲は40点から100点へと移行した。得点分布は高得点側に集中しており、全体として防災設備の位置理解および空間的関係性の把握が向上したことが確認された。

これらの結果から、本研究で構築した Minecraft ワールドを用いた体験型ゲームを通じて、参加者の防災理解度が向上したことが示された。


**5.Education Edition におけるワールド利用の実現と公開**

本研究で再現した建物内部空間について、
Minecraft JAVA Edition で制作したワールドデータを、
Minecraft Education Edition 上で利用可能な状態にすることを試みた。

JAVA Edition で作成したワールドを Education Edition に直接移行することは困難であったため、
Bedrock Edition を介した複数の移行手法を検証した。
その過程において、ワールド全体の変換が正常に行われないなどの技術的課題が確認された。

これらの課題を踏まえ、ワールド全体の移行ではなく、
建築物単位でデータを切り出して配置する方法を採用した。
その結果、本研究で再現した建物内部空間について、
Minecraft Education Edition 上で問題なく表示・移動できる状態を構築することができた。

また、Education Edition 上で利用可能なワールドデータを.mcworld 形式で作成し、GitHub の Releases 機能を用いて公開した。
当該ファイルをインポートすることで、第三者が Minecraft Education Edition 上で本研究のワールドを利用できることを確認した。

## Discussion

**1. 建物再現と防災理解の関係**

本研究では、防災教育に利用することを前提として、Minecraft 上に青山学院大学相模原キャンパスB棟を再現し、体験型ゲーム空間として構築した。建物再現にあたっては、実際の建物構造を忠実に再現すること自体を目的とするのではなく、防災行動に必要となる空間情報を学習者が理解しやすい形で提示することを重視した。

その結果、ゲーム体験の前後で実施した防災理解度テストにおいて、平均得点および中央値が大きく向上したことが確認された。これは、参加者がゲーム空間内を実際に移動しながら建物構造や防災設備の位置関係を体験したことにより、平面図や文章による説明のみでは把握しにくい「位置関係」や「動線」といった空間的理解が促進された可能性を示している。

このことから、防災教育を目的とした建物再現においては、体験的に空間を把握できるゲーム環境を構築すること自体が、学習効果を高める有効な手法であると考えられる。

**2. 防災教育を前提とした再現範囲の妥当性**

本研究では、建物内部のすべてを忠実に再現するのではなく、防災教育に必要と考えられる空間要素（主要廊下、非常階段、避難動線、防災設備の位置）を優先的に再現し、教室内装や細かな備品については抽象化を行った。
防災理解度テストの結果を踏まえると、防災設備の位置関係や避難経路に関する理解は向上しており、防災教育を目的とする場合、建物内部の完全再現が必ずしも必要ではない可能性が示された。このことから、用途を明確に設定したうえで再現範囲を取捨選択することが、効率的なゲーム空間構築につながると考えられる。

**3. 防災設備の視覚的強調表現の有効性**

本研究では、防災設備の位置をゲーム空間内で把握しやすくするため、消火器および非常階段に対して視覚的に強調したテクスチャ表現を導入した。これにより、防災設備が周囲の構造物と区別され、学習者が意識的に認識しやすい表現となった。

理解度テストにおいて、防災設備の配置や位置関係に関する設問の正答率が向上していることから、視覚的に強調された表現が学習者の注意を引き、記憶に残りやすかった可能性が考えられる。ただし、本研究ではテクスチャ表現と学習効果の因果関係を厳密に検証したわけではないため、今後の検討課題といえる。

**4. Java Edition におけるミッション実装とその前提**

本研究で実施した体験型イベントは、Minecraft Java Edition 上で構築されたワールドを用いて行われた。Java Edition では、コマンドやデータパックを用いることで、ゲーム中に特定の条件を発生させたり、行動に応じた処理を行うといった比較的高度なミッション設計が可能である。そのため、共同研究として実施したイベントでは、鬼ごっこ形式の中に複数のミッション要素を組み込んだ構成が採用された。

これにより、参加者に対してゲームとしての緊張感や目標を与えつつ、建物内を広く移動させる環境を実現することができた。しかし、これらのミッション設計は、Java Edition における技術的な自由度を前提としたものであり、そのまま他の実行環境へ移行できるとは限らない。

**5. Education Edition における技術的制約と課題**

Minecraft Education Edition では、ワールド内の挙動を制御する手段としてビヘイビアーパックの利用が可能であるものの、Java Edition におけるデータパックやコマンド設計と同一の実装を行うことは現実的に難しい。そのため、Java Edition で実現したミッションを完全に再現するには、高い技術的ハードルが存在すると考えられる。

教育現場での活用を想定した場合、複雑な設定や専門的な知識を必要とするイベント構成は、実践の妨げとなる可能性がある。この点は、本研究における今後の課題として整理されるべきである。そのため、Education Edition 上で同様の体験型防災教育を実施する際には、Java Edition における地震発生や行動判定といった複雑なミッション制御を再現するのではなく、それらの機能を簡易的な方法で代替することが有効であると考えられる。
例えば、イベント開始時にチャットを用いて「地震が発生した」という状況を一斉に提示し、あらかじめ設定した教室やフロアを安全地帯として説明することで、参加者に避難行動を促すことができる。また、建物内に配置した看板や表示物によって行動目標を示すことで、コマンドによる判定を用いずとも、鬼ごっこ形式のイベントを成立させることが可能である。

**6. ミッション要素の簡略化と防災教育への応用可能性**

以上の課題を踏まえると、防災教育の実践において重要なのは、ミッションの複雑さそのものではなく、学習者が建物内を主体的に移動し、防災設備や避難経路の位置を体験的に理解する機会を確保することである。
そのため、Education Edition 上で同様の体験型防災教育を実施する際には、Java Edition における地震発生や行動判定といった複雑なミッション制御を再現するのではなく、それらの機能を簡易的な方法で代替することが有効であると考えられる。
例えば、イベント開始時にチャットを用いて「地震が発生した」という状況を一斉に提示し、あらかじめ設定した教室やフロアを安全地帯として説明することで、参加者に避難行動を促すことができる。また、建物内に配置した看板や表示物によって行動目標を示すことで、コマンドによる判定を用いずとも、鬼ごっこ形式のイベントを成立させることが可能である。

## Conclusion

本研究では、防災教育への活用を前提として、青山学院大学相模原キャンパスB棟を Minecraft 上に再現する過程に着目し、実空間をゲーム空間へ再構成する際の課題および設計上の判断プロセスを整理した。
PLATEAU データの活用を試みたものの、変換時の不具合により手作業での再現へと方針を転換し、現地調査や写真・動画資料をもとに建物空間を構築した。その結果、防災教育に必要な空間要素を優先的に再現し、用途に応じて抽象化を行うという再現手法を示した。
また、再現した建物を用いた体験型ゲームを実施した結果、防災設備の位置や避難経路に関する理解度が向上することが確認された。これにより、防災教育を目的とした場合、建物内部の完全な再現ではなく、「何を理解させたいか」に基づいた再現設計が重要であることが示唆された。
今後は、本研究で作成した Minecraft ワールドをオープンデータとして公開することで、他の教育現場や研究においても活用されることが期待される。また、異なる建築物や利用目的に応じた再現手法の検討を進めることで、ゲーム空間を用いた防災教育の可能性をさらに広げていくことができると考えられる。
